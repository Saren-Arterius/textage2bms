# textage2bms
Convert textage.cc to BMS

# Dependencies
- Selenium
- PhantomJS
- PyQuery

# Usage
python3 textage2bms.py 'http://textage.cc/score/24/marenect.html?1AC0' > marenect.bms

# Options
- LN_DISABLE = [True/False] to disable handling of LN (textage2bms.py)

# Not supported
- BPM Change
- DP
